List Of Tags In HTML:-

  1. <p></p> Paragraph Tag:-In html,Paragraph Tag is used for paragraph writing.It gaves us the proper structure and layout to the webpage. It tells the browser that the paragraph tag has been rendered as its paragraph. Also we can add styling into paragraph tag.

  2. <a></a> Anchor Tag:-In Html,anchor tag used for to create hyperlinks to connect the webpages and we can also add telephone number and email id with the help of anchor tag .The link of webpage will be written in href section.There are many attributes which are used in anchor tag like blank,parent etc and these attributes will be written in target section into the anchor tag also there are one more attribute in anchor tag which is rel and basically anchor tag is inline element.

  3. <img></img> Image Tag:-In html,image tag have used for adding the images on webpage.Copying the image address from image then we have to put into that address in src section. In image tag there was a alt section also given for adding some information about the image basically sometimes if there was a problem in image or in image address so it doesn't rendered on webpage so alt shows the information about the image.
  4. <sup></sup> Superscript Tag:- In html,the superscript tag has used for writing the formulas or theorums.Superscript text shows in small size text. for ex:-

    a2+b 2+c 2

    This is how superscript tag works.
  5. <sub></sub> Subscript Tag:- In subscript tag text will be written below the words.Basically this tag uses for writing the formulas.This is how subscript tag works:-

    H2O CO2

  6. <mark></mark> Mark Tag:-In html,mark tag uses for highlighting purposes.The highlightened colour text showed in yellow colour.The mark tag uses for marking the specific text.For ex:-

    Lorem ipsum dolor sit amet consectetur adipisicing.

  7. <head></head> Head Tag:-In html,the head tag uses for store the meta information which rendered on browser but in head tag the meta information which was writtened doesn't shows anything on browser and we can also do styling in head tag.
  8. <u></u> Underline tag:- In html,the underline tag basically used for gave the underline to below the text.This is how underline tag works:-

    Lorem, ipsum sit amet consectetur adipisicing.
