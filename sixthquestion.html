  1. Hyper Text Markup Language(HTML)

    Primary Use Cases Key Features/Benefits
    Html is used for gave the structure and layout
    so that browser can easily rendered the content.
    We can make webpage with the help of html.
    1. HTML stands for hyper text markup language.
    2. HTML is a markup language it was platform independent language.
    3. HTML defines the structure and layout of the webpage.
    4. We can make webpage with the help of HTML.
    5. HTML can support image,audio and video.
    6. It is a simple and easy to learn language.

    In CSS we can colour our webpage or colouring the text and there are many
    more functionalities given in CSS.

    2. CSS is a stylesheet language.

    4. In CSS, we can colour the webpage or colour the text also we can increase /decrease the size.
    5. CSS is incomplete without html they both are vice vice-versa
    6. We can also write internal or external or inline css.
