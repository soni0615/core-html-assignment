Web Design

Web designing basically works on designing/creation the website.
There are many tools available for web designing like adobe photoshop etc.Web designing works on the layout of webpage it doesn't works on the functionality of webpage.Web designing works on the frontend side.Web designing works on the static and dynamic both of the websites. Web designing is the process of planning a good website or makes a good layout and structured webpage.Web designing works on UI(user interface) and UX(user experience).In web designing there are many tools which are useful for web designers some of them are:
Wordpress,Wix,shopify etc.
There are three main types of web designing:
1.Static Web Design.
2.Dynamic Web Design.
