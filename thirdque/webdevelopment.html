Web Development

Web Development made up of two words:
Web and Development Web can be basically web apps or websites and Development can be process or making of web apps or websites.
Web Development is the creating,building and maintaining the websites.
Websites can be divided into two types:
1.Static Website and another one is 2.Dynamic Website
1. Static Website: static websites are those websites which are static in nature means we cant change the content on that content stays the same.for ex:websites .
2.Dynamic website: dynamic website is always change dynamically means the content doesnt stays same on that. for ex:web apps.
Web development consists two parts: frontend and backend.
In frontend we have to learn html,css and javascript.
In backend we have to learn one language and databases .
